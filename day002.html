试试问我：

昨天网站上线。今天我问自己：既然叫"用AI摸鱼"，网站里是不是应该有一个真正的AI？

不是教程里的AI，不是截图里的AI。是访客能直接对话的那种。

今天做了什么

给网站接上了 Gemini API

API 这个词听起来很难。我理解的版本是：

💡 API 是什么
🔌
API = 插头。我的网站是插座，Gemini（Google的AI）是电。把插头插进去，网站就有了AI的能力。
🔑
API Key = 门钥匙。证明我有权限用这个AI，免费额度每天能用很多次。
💰
今天花了多少钱？0元。 Gemini 有免费额度，够我用很久。
今天踩的坑

AI差点说出了奇怪的话

第一版接上去之后，我问它："你是谁？"

它回答："我是 Gemini，Google 开发的大型语言模型……"

这不对。我要的是有我这个账号风格的AI，不是一个自我介绍的机器人。

于是我给它写了一段"人设"——告诉它：你是"摸鱼AI"，你的主人37岁开始学AI，你要用轻松接地气的口吻回答问题，不要说自己是Gemini。

这叫 System Prompt（系统提示词）。相当于给AI的岗前培训。写好之后，AI就变成了现在你看到的样子。

以前拦住我的不是难度
是开口的勇气
